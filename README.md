# FastFeast-Food-Delivery-App

1. We have created a working Full Stack website for this project. We have created this Full 
Stack Project. (React for Frontend and Node Express Js for Backend) and MySQL for 
the Database. 
2. There will be 2 folders named food-delivery-fe and food-delivery-be which will contain 
the files for frontend and backend respectively. 
3. We hope the dump of our sql db would be imported and loaded into MySQL Workbench. 
4. For Testing our application do the following: 
a. For food-delivery-be (backend code) just unzip the folder and open the folder in 
another instance of any code editor (lets say vscode). 
b. If you don’t have nodejs installed please install nodejs from 
https://nodejs.org/en/download. 
c. In the terminal please do a “npm install” command to install the 
node_modules(which we will have deleted to reduce the size of the file). 
d. Before doing the “npm run server” command please go into the .env file in the 
root and fill in the details such as DB_HOST=localhost, DB_USER, 
DB_PASSWORD, DB_NAME=food_delivery, PORT=3000, 
JWTSECRET=”securekey” 
e. Please enter the tester’s db_user(by default root) and password in 
DB_PASSWORD. 
f. 
After the node packages are installed type in the command “npm run start” or 
“npm run server” to start the node js application which will start in port 3000. (Can 
be accessed at localhost:3000 locally - Dont have to access this as this is just a 
node server - just let it run in the background). 
g. For food-delivery-fe just unzip the folder and open the folder in any code editor 
(lets say vscode). 
h. In the terminal please do a “npm install” command to install the 
node_modules(which we will have deleted to reduce the size of the file). 
i. 
After the node packages are installed type in the command “npm run dev” to start 
the react app which will start in port 5173. (Can be accessed at localhost:5173 
locally). 
j. 
This would setup the frontend up and running in the local machine. 
k. At this point you’d be able to use the application.
